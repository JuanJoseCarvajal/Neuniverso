# Guía de Contribución — NEUNIVERSO

Gracias por tu interés en contribuir a NEUNIVERSO.

## Reglas generales
- Toda la documentación, comentarios y experiencia de usuario deben estar en español.
- Sigue las mejores prácticas enterprise-grade para frontend, backend, DevOps y seguridad.
- Mantén la coherencia en la arquitectura y el diseño del sistema.

## Flujo de trabajo
1. Crea un fork del repositorio.
2. Crea una rama descriptiva para tu feature o fix.
3. Realiza tus cambios siguiendo la estructura y convenciones del monorepo.
4. Asegúrate de que los tests pasen y el código esté formateado.
5. Abre un Pull Request detallando tus cambios.

## Código de conducta
- Sé respetuoso y profesional.
- Toda comunicación y documentación debe ser en español.

¡Bienvenido a la arquitectura de nuevas realidades!
